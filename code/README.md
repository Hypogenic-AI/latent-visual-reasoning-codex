# Cloned Repositories

## Repo 1: relational-networks
- URL: https://github.com/kimhc6028/relational-networks
- Purpose: Baseline relational reasoning model; includes Sort-of-CLEVR generator.
- Location: `code/relational-networks/`
- Key files: `code/relational-networks/main.py`, `code/relational-networks/model.py`, `code/relational-networks/sort_of_clevr_generator.py`
- Notes: Useful baseline for relational abstraction over object representations.

## Repo 2: clevr-dataset-gen
- URL: https://github.com/facebookresearch/clevr-dataset-gen
- Purpose: Official CLEVR dataset image and question generation.
- Location: `code/clevr-dataset-gen/`
- Key files: `code/clevr-dataset-gen/image_generation/`, `code/clevr-dataset-gen/question_generation/`
- Notes: Use to generate or inspect CLEVR-style synthetic scenes.
